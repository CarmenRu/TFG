\capitulo{2}{Objetivos}

\section{Objetivo general}
Desarrollar una herramienta capaz de detectar posibles interacciones farmacológicas entre principios activos mediante el análisis de las enzimas que las metabolizan.

\section{Objetivos específicos}

\begin{itemize}
    \item Extraer información de DrugBank mediante procesamiento XML.
    \item Extraer información de las fichas técnicas incluidas en CIMA con la ayuda de NotebookLM y ChatGPT.
    \item Extraer información de los efectos adversos recogidos en SIDDER, distinguidos por los códigos ATC de cada principio activo.
    \item Integrar la información obtenida de las distintas fuentes en una base de datos farmacológica unificada.
    \item Diseñar un algoritmo de clasificación de riesgo.
    \item Implementar un sistema de búsqueda de alternativas terapéuticas basado en códigos ATC.
    \item Desarrollar una visualización accesible y sencilla para el usuario.
    \item Validar el funcionamiento mediante casos de prueba.
\end{itemize}
